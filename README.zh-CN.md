# TRX / ETH 靓号地址生成器

这是一个本地离线运行的 Windows 桌面工具，用来生成 **TRX / ETH 靓号地址**。重点是私钥正确、地址可导入真实钱包、任务过程不联网、不上传任何敏感数据。

[返回英文首页](./README.md) | [Release 下载](https://github.com/jacksongua8221-cell/trx-eth-vanity-address-studio/releases)

> 本地离线运行。不上传私钥。不上传助记词。不上传地址。不上传任务记录。

![主窗口截图](./docs/images/main-window.png)

## 最新功能

- 支持 TRX / ETH 两条链。
- 输出标准 `secp256k1` 私钥，可导入真实钱包。
- ETH 地址按 Keccak-256 和 EIP-55 checksum 规则生成。
- TRX 地址按主网规则生成：公钥 -> Keccak-256 -> 后 20 字节 -> `0x41` -> Base58Check。
- 支持私钥生成和助记词生成。
- 助记词模式下可选择保存私钥、助记词，或两者都保存。
- 匹配方式支持前缀、后缀、包含、前缀 + 后缀、前缀 + 包含、包含 + 后缀、前缀 + 包含 + 后缀、智能识别。
- 疑似靓号只看后缀，不看前缀和中间。
- 疑似规则支持豹子号、数字顺子、`abcde` / `edcba`、自定义后缀。
- 不再内置 `dead / beef / cafe / face / feed` 这类英文词，英文后缀由用户自定义。
- 集成“靓号筛选”标签页，可实时同步当前生成出来的疑似靓号，也支持导入 TXT。
- 筛选结果会自动分类，地址高亮命中的后缀。
- 私钥和助记词在筛选表格里默认短显示，提供复制按钮，避免长文本挤压界面。
- CPU 线程数运行中可实时调整。
- 根据当前电脑 CPU 自动给出线程建议。
- 中间仪表盘显示 CPU 型号、系统内存、GPU 名称、显存、温度、功耗、使用率。
- 生成速度使用中文显示为“地址/秒”。
- 点击开始、暂停、继续、停止会立即反馈状态。
- worker 会更快回传统计，避免大批量模式下速度长时间不刷新。
- GPU 当前用于状态监控，生成任务由 CPU 多线程执行。
- 窗口支持拉动放大，布局自适应。

## 保存逻辑

目标命中结果保存到：

```text
results/results.txt
```

疑似靓号结果保存到：

```text
results/suspicious/suspicious.txt
```

TXT 每行只保存用户选择的内容，例如：

```text
地址 私钥
地址 私钥 助记词
```

疑似靓号不会混到右侧目标命中列表里，会单独保存并可在筛选页查看。

## 疑似靓号规则

只看后缀：

```text
Txxxxxx8888  -> 命中
T8888xxxxxx  -> 不命中
Txxx8888xxx  -> 不命中
```

支持：

```text
后缀豹子号：8888 / 9999 / aaaa / 7777777 / 88888888
后缀数字顺子：12345 / 234567 / 98765 / 876543
后缀字母顺子：abcde / edcba
自定义后缀：用户自己输入，一行一个
```

## 验证

```bash
npm install
npm test
npm run verify:offline
```

测试内容包括：

- 10 个 ETH 私钥和地址用 `ethers` 校验。
- 10 个 TRX 私钥和地址用 `TronWeb` 校验。
- 私钥导入格式校验。
- 助记词生成校验。
- 加密保存和错误密码拒绝校验。
- checkpoint 恢复校验。
- 疑似靓号后缀规则校验。
- 离线源码扫描，检查是否存在直接网络 API。

## 风险提示

生成出来的私钥和助记词就是实际钱包资产凭证。不要把结果文件、私钥、助记词、带私钥截图发给任何人。正式使用前建议先用空钱包导入验证。
