# TRX / ETH 靓号地址生成器 - 中文说明

这是一个 Windows 本地离线桌面工具，用来生成 TRX / ETH 靓号地址。项目现在保留两个版本：功能完整的标准桌面版，以及专门追求速度的 GPU 极速版。

## 一句话说明

你输入想要的地址规则，比如：

```text
TRX 后缀 888888
TRX 后缀 666666
ETH 后缀 deadbeef
ETH 前缀 1234
```

软件会在本机不断生成随机钱包地址，命中规则后保存地址和私钥。没有命中的地址不会保存，也不会一直堆在内存里。

## 版本选择

### 标准桌面版

标准版适合日常使用，功能比较全：

- 支持 TRX / ETH
- 支持私钥生成
- 支持助记词生成
- 可以选择只保存私钥、只保存助记词、两个都保存
- 支持前缀、后缀、包含、前缀 + 后缀等匹配方式
- CPU 多线程生成，线程数可调
- GPU 状态监控：显卡名称、使用率、显存、温度、功耗
- 显示生成速度、尝试次数、理论难度、累计命中概率、预计耗时
- 自动保存，防止中途退出后结果丢失
- 带靓号筛选页面，可导入 TXT，也可实时同步疑似靓号

### GPU 极速版

GPU 极速版适合只跑目标靓号，界面更简单，保存更直接：

- TRX GPU：基于 OpenCL GPU 核心
- ETH GPU：基于 OpenCL GPU 核心
- 一行一个目标后缀，可以同时跑多个目标
- TRX 后缀会按长度分组，例如 5 位、6 位、7 位分开启动，避免不同长度混跑导致命中逻辑不准
- 目标命中保存到 `target.txt`
- 疑似豹子 / 顺子保存到 `suspicious.txt`
- 不保存未命中的普通地址

## GPU 极速版疑似规则

疑似只看后缀，不看前缀，不看中间。

豹子规则可自己设置最少位数：

```text
5 位起：88888 / aaaaa
6 位起：888888 / bbbbbb
7 位起：7777777 / ccccccc
8 位起：99999999 / dddddddd
```

数字豹子和字母豹子可以单独开关。

顺子只保留下面这些指定后缀：

```text
123456
1234567
12345678
1234567890
012345
987654
654321
```

不会内置 `dead / beef / cafe / face / feed` 这种英文词。如果你想要英文后缀，自己填到目标内容里即可。

## 结果文件

标准桌面版默认结果目录：

```text
results/
```

GPU 极速版默认结果目录：

```text
靓号生成器_便携版/resources/app/results/
```

目标命中：

```text
target.txt
```

疑似结果：

```text
suspicious.txt
```

TXT 每行格式尽量保持简单：

```text
地址 私钥
```

## 钱包兼容性

ETH：

```text
私钥 -> secp256k1 公钥 -> Keccak-256 -> 后 20 字节 -> 0x 地址
```

TRX：

```text
私钥 -> secp256k1 公钥 -> Keccak-256 -> 后 20 字节 -> 加 0x41 -> Base58Check -> T 开头地址
```

私钥是 32 字节，也就是 64 位十六进制字符。部分钱包导入 ETH 私钥时只接受 64 位，不接受 `0x` 前缀；如果钱包提示长度错误，把前面的 `0x` 去掉即可。

## 从源码运行标准版

```bash
npm install
npm start
```

验证：

```bash
npm test
npm run verify:offline
```

打包：

```bash
npm run package:portable
```

## 从源码运行 GPU 极速版

源码目录：

```text
gpu-edition/
```

运行方式：

```bash
cd gpu-edition
npm install
npm start
```

注意：GPU 版需要 Windows、NVIDIA 驱动、OpenCL 环境。不同显卡、驱动和目标规则会影响速度。

## 安全说明

- 本工具本地离线运行，不需要联网。
- 不要把私钥、助记词、结果 TXT、带私钥截图发到 GitHub 或聊天软件。
- 跑出来的地址和私钥是一对，导入钱包前先小额测试。
- 靓号越长越难，6 位、7 位、8 位尾号需要更长时间。
- 本项目不是破解钱包工具，不能通过地址反推别人的私钥。
