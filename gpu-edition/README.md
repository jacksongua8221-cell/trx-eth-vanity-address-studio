# GPU 极速版

这个目录是 TRX / ETH GPU 靓号生成器的公开源码包，保留程序源码和 GPU 核心文件，不包含运行结果、私钥、runtime 缓存和 Electron 便携版外壳。

## 功能

- TRX OpenCL GPU 目标地址生成
- ETH OpenCL GPU 目标地址生成
- 只保存目标命中和疑似命中
- 支持一行一个目标后缀
- TRX 后缀按长度分组运行
- 疑似豹子可设置 5 位起、6 位起、7 位起等
- 数字豹子、字母豹子可单独控制
- 顺子只保留指定后缀：`123456`、`1234567`、`12345678`、`1234567890`、`012345`、`987654`、`654321`

## 运行

```bash
npm install
npm start
```

结果默认保存到：

```text
results/target.txt
results/suspicious.txt
```
